# Access Control System

A robust and scalable Access Control and User Management System built with **Laravel 12**. This application provides an intuitive interface for managing users, roles, and granular permissions using the powerful `spatie/laravel-permission` package. The system features a modern, responsive UI built with **TailwindCSS** and **Blade Components**, fully localized in Arabic.

---

## 🚀 Features

* **Secure Authentication & Authorization:** Robust login and session management to ensure your application remains secure.
* **Comprehensive User Management:** Full CRUD (Create, Read, Update, Delete) operations for users, including role assignments directly from the UI.
* **Advanced Role Management:** Create and edit roles, and assign multiple permissions to any role seamlessly.
* **Granular Permission Control:** Define specific application permissions and manage them efficiently.
* **Spatie Laravel-Permission:** Fully integrated with the industry-standard package for RBAC (Role-Based Access Control) in Laravel.
* **Modern UI/UX:** A responsive, clean dashboard designed with TailwindCSS.
* **Arabic Localization:** Fully translated interface, validation messages, and flash notifications for Arabic-speaking users.

---

## 🛠️ Tech Stack

* **Backend:** [Laravel 12](https://laravel.com/) (PHP 8.2+)
* **Frontend:** Laravel Blade, [TailwindCSS 4](https://tailwindcss.com/), Vite
* **Database:** MySQL / SQLite

## ⚙️ Installation & Setup

Follow these steps to get your development environment up and running:

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd roles-permissions-control-app
   ```

2. **Install PHP Dependencies**
   ```bash
   composer install
   ```

3. **Install NPM Dependencies**
   ```bash
   npm install
   ```

4. **Environment Configuration**
   Copy the example environment file and set your configuration variables (database credentials, etc.).
   ```bash
   cp .env.example .env
   ```

5. **Generate Application Key**
   ```bash
   php artisan key:generate
   ```

6. **Database Migration & Seeding**
   Ensure your database is created and configured in `.env`, then run migrations along with the default seeders to populate initial roles, permissions, and an admin user.
   ```bash
   php artisan migrate --seed
   ```

7. **Compile Frontend Assets**
   Build the TailwindCSS assets using Vite.
   ```bash
   npm run build
   # or for active development: npm run dev
   ```

8. **Run the Application**
   Start the Laravel local development server.
   ```bash
   php artisan serve
   ```
   *The application will be accessible at `http://localhost:8000`.*

## 📖 Usage

1. **Login:** Navigate to the root URL `/` or `/login` and authenticate using the credentials generated by the `UserSeeder`.
2. **Dashboard:** Upon successful login, you will be redirected to the dashboard.
3. **Manage Access:** Use the navigation menu to access:
   * **Users (`/users`):** Manage user accounts and assign roles.
   * **Roles (`/roles`):** Define roles and attach specific permissions.
   * **Permissions (`/permissions`):** View and manage available system permissions.
